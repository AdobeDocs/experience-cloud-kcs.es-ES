# Alternativa para la actividad Recolector de archivos en ACS

## Descripción {#description}


<b>Entorno</b>

Adobe Campaign Standard

<b>Problema/Síntomas</b>

Como ACS no tiene la actividad Recolector de archivos , puede crearla con las siguientes actividades para lograr funcionalidades similares.


## Resolución {#resolution}


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)

La transferencia de archivos permitirá descargar varios archivos según el patrón de nombre de archivo de un servidor externo. Cada archivo se pasa a la actividad Cargar archivo , lo que proporciona datos a la actualización.

Una vez finalizado, continúa con el siguiente archivo, etc.

La transferencia de archivos también tiene la configuración de histórica, tal como se encuentra en el Recopilador de archivos.
