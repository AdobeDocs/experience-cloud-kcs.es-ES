# Cómo exportar las referencias locales de un activo en formato JSON

## Descripción


Puede ver las referencias locales de un bien en la pestaña [!UICONTROL Referencias] de [!UICONTROL Propiedades]. (En la consola [!DNL Assets], seleccione un activo, seleccione [!UICONTROL Propiedades] en la barra de herramientas y, a continuación, seleccione la pestaña [!UICONTROL Referencias]). Si usa la información fuera de AEM, puede exportarla en formato JSON.

<b>Entornos</b>

AEM as a Cloud Service, AEM 6.5


## Resolución


Al acceder a la siguiente URL&#39;1&#39; se obtienen referencias locales en formato JSON. Los valores `path` en la respuesta son las rutas de contenido que hacen referencia a un activo de destino.

&#39;1&#39; http://localhost:4502/bin/wcm/references.json?path=path al activo objetivo

<b>Ejemplo de uso:</b>
http://localhost:4502/bin/wcm/references.json?path=/content/dam/wknd/en/activities/hiking/equipment_6.jpg
