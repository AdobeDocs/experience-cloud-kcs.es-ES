# Error: org.apache.sling.api.request.TooManyCallsException en la consola personalizada


El artículo describe cómo resolver el problema de AEMaaCS en el que la variable *org.apache.sling.api.request.TooManyCallsException* El error aparece en la consola personalizada. Puede superponer la ruta:` …/aem/apps/lv/console/lvjobs/content/lvjobs.html` O establezca el límite globalmente utilizando la propiedad `'sling.max.record.requests'` en el PID de configuración de OSGi `org.apache.sling.engine.impl.SlingMainServlet`.
