# Campaign Classic: El proceso WEB informa de los archivos JAR que faltan al reiniciar

<b>Causa</b>
El proceso WEB informa de algunos archivos JAR (archivo Java) que faltan.


## Resolución {#resolution}


El `classpath` etiqueta de la `serverConf.xml` el archivo de configuración no se ha actualizado correctamente después de la actualización.

El `classpath` debe señalar a los JAR más recientes.
