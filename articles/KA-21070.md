# Copia de MID en una eVar o una propiedad en el SDK móvil de AEP

## Descripción {#description}

Pasos para copiar MID en una eVar o una propiedad para el SDK móvil de AEP

## Resolución {#resolution}


- &#x200B;&#x200B;&#x200B;&#x200B;&#x200B;&#x200B;&#x200B; &#x200B; &#x200B; &#x200B; &#x200B; &#x200B;Configure el elemento de datos en Adobe Launch para recuperar el valor MID. 
   - Configuración:
      - Nombre- ECID
      - Extensión: Mobile Core
      - Tipo de elemento de datos: ID&#x200B;&#x200B;&#x200B;&#x200B;&#x200B;&#x200B; del Experience Cloud
- Cree una regla de Launch para enviar el MID en forma de datos de contexto.
   - Configuración:
      - Eventos: Extensión: Mobile Core, Tipo de evento: Estado de seguimiento<b> O</b>
      - Extensión: Mobile Core, Tipo de evento: Rastrear acción
      - Acciones: Extensión, Mobile Core, Tipo de acción, Adjuntar datos
      - Carga útil JSON- 
         - ![](assets/3a13db8a-e6b8-ed11-83fe-6045bd0065f9.png)
- Cree una regla de procesamiento en Analytics para copiar el valor de MID en un eVar o una propiedad.
